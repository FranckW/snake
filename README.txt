Listing des dossiers et fichiers du projet
	- classes : dossier contenant les fichiers .class du projet
	- doc : dossier contenant la Javadoc du projet
	- src : dossier contenant les fichiers source du projet
	- manifest : manifest du projet
	- plan.jpg : explication en image du jeu
	- principe.txt : principe du jeu (affiché au lancement de l'archive)
	- mysnake.jar : archive exécutable du jeu

Instruction :
	- se placer à la racine du projet (dossier snake) et lancer la commande : java -jar snake.jar
